### 首次 Git 提交前：设计探索与交互原型

- 参考用户提供的页面截图和两段录屏，确认目标不是 React Bits 的 Card Swap，而是上下翻转、主卡背后不堆叠内容的纵向卡片队列。
- 调研 Motion Sites 等动效参考，并使用用户提供的 3D cylinder carousel 提示词作为卡片运动基础。
- 使用 React Bits Soft Aurora 思路和 OGL shader 创建黑色背景上的粉、紫、蓝动态光带。
- 建立第一页英文标题 `NANA AUDIOVISUAL ART CREATION`、顶部导航、辅助线和课程卡片区域。
- 删除卡片上的数字、文字、磁卡芯片等装饰，课程内容改为 GIF/动态图占位。
- 建立卡片悬停标题、点击主卡打开课程、悬停上下卡自动居中、滚轮驱动和 3 秒自动轮播。
- 卡片数量采用有限可见窗口，页面只显示约 6 张附近卡片，但可循环访问全部课程。
- 处理非全屏缩放、卡片圆角闪烁、背面内容、边缘细线和标题标签换行。
- 将首页介绍文案更新为：`Explore the relationship between what you see and what you hear. Build your own art worlds.`
